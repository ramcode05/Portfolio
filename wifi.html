Fuzzy Matching & Natural Language Parsing

The Best Feature: Intelligent Parsing

Unlike standard command-line tools that crash or fail on minor syntax deviations, this toolkit is engineered with an autonomous correction engine.

  • Natural Language Input: Interacts using flexible phrasing like find wifi, search ports, or who is connected instead of rigid, hard-to-remember syntax codes.
  • Auto-Correction for Typos: Leverages mathematical string distance algorithms to verify inputs against a localized database, correcting typos in real-time for zero-interruption execution.
  • Synonym Normalization: Maps general vocabulary verbs like locate, find, leaks, and probe directly to their underlying Python system call commands.

Passive Audit Commands

> password Inspects and extracts cleartext passwords for all saved Wi-Fi profiles stored locally on the host OS.
> analyse network Triggers passive local beacon scans, logging nearby SSIDs, signal decibel strength, and security protocols.
> detect mitm Passively sniffs and cross-references active ARP tables to identify active Man-in-the-Middle spoofing attacks.
> optimize Calculates overlapping channel noise to recommend the cleanest Wi-Fi broadcasting channel.
> check <email> Performs breach lookup searches to check if email addresses have been compromised in data leaks.

Active & Diagnostic Commands

> scan ports Fires rapid TCP socket connectors to perform an intense Nmap-style scan of top ports on target IPs.
> who is connected Sweeps the current subnet using active ARP queries, displaying device MACs, IP leases, and vendor details.
> geo trace <host> Tracks network route hops and attempts to geolocate routers across routing nodes.
> subdomain <domain> Discovers hidden web hosts and subdomains by querying a preloaded dictionary of standard endpoints.
> export html Compiles active session logs and exports a structured, styled HTML analysis dashboard.

Fuzzy Matching Engine Source

The core algorithm powering the toolkit's user-friendly prompt. Located in main.py, it uses difflib.get_close_matches to evaluate typo distances and replaces synonyms on the fly before commands are routed to the executor:

main.py > fuzzy_match_command 
def fuzzy_match_command(cmd):
    """Corrects common typos and normalizes synonyms."""
    from difflib import get_close_matches
    
    # Core technical keywords mapped to system logic
    core_commands = [
        "password", "analyse", "analyze", "network", "scan",
        "port", "ports", "who", "connected", "site", "router",
        "default", "defaults", "sniff", "wake", "whois",
        "honeypot", "detect", "mitm", "optimize", "gen",
        "pass", "dns", "enum", "backup", "wifi", "track",
        "check", "geo", "trace", "subdomain", "speedtest",
        "speed", "monitor", "export", "html", "help",
        "clear", "exit", "quit", "my", "ip"
    ]
    
    # Synonym mapping for intuitive Natural Language input
    synonyms = {
        "find": "scan",
        "search": "scan",
        "probe": "scan",
        "list": "show",
        "show": "show",
        "networks": "network",
        "wifi": "network",
        "connections": "connected",
        "devices": "connected",
        "speed": "speedtest",
        "internet": "speedtest",
        "location": "track",
        "locate": "track",
        "where": "track",
        "breach": "check",
        "leaks": "check",
        "hack": "scan"
    }
    
    all_keywords = core_commands + list(synonyms.keys())
    words = cmd.lower().split()
    corrected = []
    
    for word in words:
        # 1. Exact match check
        if word in all_keywords:
            corrected.append(word)
            continue
            
        # 2. Typos correction (75% similarity threshold)
        matches = get_close_matches(word, all_keywords, n=1, cutoff=0.75)
        if matches:
            corrected.append(matches[0])
        else:
            # Keep arguments (IPs, domains) as-is
            corrected.append(word)
            
    # 3. Apply Synonym Replacements
    final_cmd_words = []
    for word in corrected:
        if word in synonyms:
            final_cmd_words.append(synonyms[word])
        else:
            final_cmd_words.append(word)
            
    return " ".join(final_cmd_words)
